# cpu-4bits-logisim

Proyecto de CPU básica de 4 bits desarrollado en Logisim Evolution.

## Características

- Arquitectura de 4 bits
- Unidad lógica y aritmética (ALU)
- Memoria y registros
- Sistema de control

## Archivos para poder ver y ejecutar la simulación 

- **cpu-4bits.circ**: Diseño del CPU en Logisim Evolution
- **Instalador**: Software necesario para ejecutar el proyecto
